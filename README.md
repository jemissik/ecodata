# ECODATA-Prepare

ECODATA-Prepare is a set of Python-based apps to access and process remote sensing and other environmental data products and prepare them for further use along with animal movement data. ECODATA-Prepare is designed to be along with the ECODATA-Animate tool to create movies of animal movement data, and can also be used to create input to resource, habitat and step selection models.
